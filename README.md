# 🏦 SH ATM (Web-Based ATM Simulator)

A modern, responsive ATM simulation web app built using **HTML, CSS, and JavaScript**.  
It mimics real ATM functionality including PIN setup, deposits, withdrawals, balance checking, and transaction history — all stored locally in the browser.

---

## ✨ Features

- 🔐 Secure PIN setup & validation
- 💰 Deposit cash functionality
- 💸 Withdraw cash with balance checks
- 📊 Live balance tracking
- 🧾 Transaction history (last 20 records)
- 💾 Data persistence using `localStorage.`
- 📱 Fully responsive UI (mobile + desktop)
- 🎨 Modern dark-themed fintech UI
- ⚡ Instant updates without page reload

---

## 🖥️ Demo Pages

- **Dashboard** → Overview of balance, stats, and recent transactions  
- **Set PIN** → Create or change your ATM PIN  
- **Deposit** → Add money to your account  
- **Withdraw** → Withdraw money with validation  
- **Balance Check** → View current balance securely  
- **History** → View full transaction logs  

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom UI Design + Responsive Layout)
- Vanilla JavaScript (No frameworks)
- LocalStorage API
